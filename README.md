# Fitness App Dashboard

A modern, responsive fitness tracking dashboard built with React, Vite, and Tailwind CSS.

## Features

- 🏃‍♂️ Activity tracking and visualization
- 📊 Interactive charts and analytics
- 🎯 Goal setting and progress monitoring
- 📱 Responsive design for all devices
- 🌙 Dark/Light theme support
- 💪 BMI calculator and health metrics

## Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Utilities**: CLSX, Tailwind Merge

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/surajsm31/Fitness-App-Dashboard.git
cd Fitness-App-Dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
